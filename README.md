# Projeto SQL - Banco de Dados E-commerce #

Projeto desenvolvido com o objetivo de praticar conceitos de Banco de Dados Relacional utilizando SQL, abrangendo desde consultas simples até consultas avançadas com múltiplos relacionamentos.

---

## Objetivo

Este projeto tem como finalidade aplicar na prática os principais comandos SQL utilizados no desenvolvimento de sistemas, simulando um banco de dados de um **E-commerce**.

Durante o desenvolvimento foram utilizados recursos como:

- Modelagem Relacional
- Chaves Primárias e Estrangeiras
- Integridade Referencial
- CRUD Completo
- JOINs
- Subqueries
- Funções de Agregação
- EXISTS e IN
- Agrupamentos
- Ordenações
- Relatórios

---

# Estrutura do Banco

O banco foi modelado para representar um sistema de vendas contendo tabelas relacionadas entre si.
